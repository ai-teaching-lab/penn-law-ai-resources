Claude Code Skills
Faculty Tools Open Source

These are open-source Claude Code skills I've built for common faculty tasks — reusable tools you invoke by name inside Claude Code. They require a paid Claude subscription and Claude Code (terminal, VS Code, or desktop app). Install the ones you want from the GitHub repo below.

01 Teaching & Assessment
MCQ Generator
Generate multiple-choice exam questions grounded in psychometric research. Distractor validation, cognitive taxonomy tagging, coverage balancing. Supports course presets.
law-mcq-generator
Essay Question Generator
Generate essay exam questions with SOLO taxonomy layering, construct alignment to course materials, and rubrics designed for AI-assisted grading.
law-essay-generator
Class Prep
Comprehensive class preparation — checks slide-to-reading alignment, reviews class problems, and produces a lecture guide document. All three in one pass.
law-class-prep
Class Problems
Create and revise adversarial in-class hypotheticals for any law course. Enforces problem-to-reading alignment and pedagogical rigor.
law-class-problems
Lecture Slide Reviewer
Review lecture slides against assigned readings for coverage, pacing, and gaps. Standalone or as part of full class prep.
lecture-slide-reviewer
02 Faculty Tools
Memo Production
Produce formatted Word (.docx) memos with Penn Carey Law letterhead — proper margins, Cambria 12pt, official logo.
law-memo
Document Production
Proposals, reports, briefing docs, white papers — formatted .docx documents in Penn Carey Law style.
law-document
Email Drafting
Draft emails and professional communications in your voice — replies, declines, invitations, follow-ups. Learns your style.
law-email-style
PDF Rendering
Convert Markdown to polished, professionally formatted PDFs in Penn Carey Law house style. Reading lists, handouts, reports.
md-to-pdf
Document Comment Summary
Extract and summarize all comments from Word (.docx) files into a clean report. Compiles reviewer feedback across multiple documents.
docx-comment-summary
Rex — Critical Reviewer
A senior engineering critic that reviews code, plans, designs, and documents. Finds problems before they ship — blunt, specific, actionable.
rex
